## 💻 Sobre o projeto


O Marriage é um projeto desenvolvido durante meus estudos em Desenvolvimentos de sistemas com o objetivo de praticar a criação de páginas web utilizando React, Next.js, typeScript, CSS e Backend. Durante o desenvolvimento, foram aplicados conceitos de semântica HTML, responsividade, organização de código e boas práticas de estruturação de interfaces, resultando em uma landing page moderna e bem estruturada.
